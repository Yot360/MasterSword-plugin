# MasterSword-plugin
A minecraft plugin that adds a master sword, a very powerfull weapon.
(My first plugin :))


## Commands

### /mastersword  
  Give the Master Sword to the player.  
  Needs `mastersword.use`
  
### /masterspawn
  Spawns a chest with a master sword in it.  
  Needs `masterspawn.use`
